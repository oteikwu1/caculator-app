# A simple web base caculator 

# A simple functional caculator built using HTML, CSS JAVASCRIPT

## Feature

*Core mathematic opeartion:* Addition, Division, Multiplication and Subtraction
*Support Decima:* Handles floating-point numbers accurately.
*Clear Function:* ('C') To clear the whole input ('Del') Delete from the last index


## Technologies Used

* **HTML5:** Structure and semantic layout.
* **CSS3:** Styling, grid/flexbox layout, and responsiveness.
* **JavaScript (ES6):** Calculation logic and DOM manipulation.

##  Project Structure


 index.html      # The visual layout and structure
 style.css       # The design, colors, and responsiveness
 script.js       # The calculation logic and button triggers
